## ✨ Features

- **⚡ Real-Time Communication**: Uses WebSockets (`ws`) to achieve low-latency, two-way messaging without page reloads.
- **🎨 Glassmorphic UI**: Beautiful, modern aesthetic interface featuring frosted-glass effects, dynamic dark-mode gradients, and smooth slide-in animations.
- **📱 Responsive Design**: Designed and crafted to work effortlessly across desktop and mobile devices.
- **🌍 Global Chat Room**: Instantly connect and chat globally with anyone logged onto the server.

## 🚀 Installation & Setup

Want to run this project locally? Follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vviqry/ws-project.git
   ```

2. **Navigate into the directory:**
   ```bash
   cd ws-project
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the server:**
   ```bash
   node index.js
   ```

5. **Open the App:**
   Open your browser and navigate to: [http://localhost:8080/chat.html](http://localhost:8080/chat.html)

## 🛠️ Tech Stack

- **Backend Architecture**: Node.js, Express.js
- **Network Protocol**: WebSockets (`ws` library)
- **Frontend Interactivity**: Vanilla JavaScript, HTML5
- **UI & Styling**: CSS3 (Custom Glassmorphism framework)
